# Funciones en Azure para interactuar con ChatBot y Table Storage 

Fecha: 2020 Agosto

Por: Luz Eunice / José Cruz

Funciones para interactuar con las tablas: 
- asistente : Información de los asistentes a las sesiones
- usuario : Colaboradores del área
- sesion : Sesiones planeadas y solicitantes a impartir las sesiones

Funciones generadas: 
- asistentes 
- solicitudes
- lausuarios
- lasesiones 
- laasistentes_cancelar


Requsitos en azure: 
- Function App
- Key vault 
- Tablas en un blob storage.
